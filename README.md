## 🗒 About

These are Currency Convertor App in which you can one country to other country currency convert and check a currency rate.

## ⚙️ Features

- Main features are as follows:

    - Splash Screen
    - Dark & Light Theme
    - Currency Convert
    - Currency API Integration
    - Currency Rate
    - Create Number Buttons
